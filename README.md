# E-Commerce Sales Analysis

## Project Overview
Analysis of 400,000+ transactions from a UK-based online retail store
to identify revenue trends, top products, and key markets.

## Dataset
- **Source:** Kaggle — Online Retail Dataset
- **Link:** https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset
- **Note:** Full dataset not uploaded due to size. A 1000-row sample
  is included in this repository as `Online_Retail_Sample.csv`

## Tools Used
- **Python** — pandas, matplotlib, seaborn
- **Power BI Desktop** — interactive dashboard
- **Jupyter Notebook** — code environment

## Key Findings
- Total Revenue: £8.91 Million across 18,530 orders
- November is the peak revenue month — nearly 3x average monthly revenue
- UK accounts for the majority of all sales
- Paper Craft Little Birdie is the top revenue-generating product
- Average revenue per customer is £2,054 suggesting wholesale buyers

## Project Steps
1. **Data Collection** — Downloaded from Kaggle
2. **Data Cleaning** — Python/pandas (removed nulls, duplicates, bad rows)
3. **Exploratory Data Analysis** — 5 charts covering trends and top products
4. **Dashboard** — Built in Power BI with KPI cards, line chart, map and bar chart
5. **Insights Report** — Written summary of findings and recommendations
